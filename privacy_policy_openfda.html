Privacy Policy: OpenFDA Repo File Fetcher GPT

Effective Date: June 13, 2025

1. Purpose and Scope

This Privacy Policy describes how the OpenFDA Repo File Fetcher GPT (“the Application”) processes, stores, and protects data. The Application is designed solely to fetch publicly available files from the FDA/OpenFDA GitHub repository.

2. Data Collected and Processed

User Inputs:

The Application temporarily processes the branch and path parameters provided by the user to construct the GitHub API request.

No other user-provided data is collected or stored.

Personal Information:

The Application does not collect any personally identifiable information (PII), such as names, email addresses, IP addresses, or similar identifiers.

No cookies, tracking scripts, or analytics tools are used.

3. External API Access and Security

Service Used:

The Application sends HTTPS GET requests to:

https://raw.githubusercontent.com/FDA/openfda/{branch}/{path}

Only publicly accessible content is fetched in read-only mode; no authentication or private repository access is employed.

Data Transmission:

All requests and responses occur over TLS/HTTPS, ensuring encryption in transit.

No data is forwarded to third parties beyond the GitHub raw content service.

4. Data Retention and Deletion

Ephemeral Processing: User-supplied parameters and the fetched file content are stored only in memory for the duration of the request.

Automatic Deletion: Once the request completes or the session ends, all data is immediately discarded.

No Persistent Storage: The Application does not write any data to disk, database, or logs.

5. Monitoring and Analytics

The Application does not implement any form of usage analytics, error reporting services, or user-activity tracking.

No metrics are recorded or shared.

6. User Rights and Contact

Access and Correction: Since no personal data is stored, there are no records to access or correct.

Data Deletion Requests: All data is transient and automatically deleted; no further action is required.
